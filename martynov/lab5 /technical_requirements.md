# Технические требования для генерации лендинга

## Общие требования
- Использовать только HTML, CSS и JavaScript (без внешних зависимостей)
- Обеспечить полную адаптивность (корректное отображение на всех устройствах)
- Код должен быть модульным, читаемым и пригодным для поддержки
- Кроссплатформенная совместимость (работа в любом современном браузере)

## Структура файлов
- `index.html` - основная HTML-структура
- `styles.css` - стили оформления
- `script.js` - интерактивные элементы (опционально)

## Функциональные требования
- Современный, профессиональный дизайн
- Минимум 6 блоков контента
- Форма обратной связи с валидацией
- Анимации и переходы для улучшения UX
- SEO-оптимизация (мета-теги, семантическая разметка)

## Дизайн-требования
- Цветовая схема: синий (#0066CC) и белый с акцентами
- Шрифты: современные, читаемые (sans-serif)
- Минималистичный стиль с акцентом на call-to-action элементы
- Высококачественные изображения (плейсхолдеры)

## Технические ограничения
- Максимальный размер файлов: HTML < 50KB, CSS < 30KB, JS < 20KB
- Поддержка браузеров: Chrome, Firefox, Safari, Edge (последние версии)
- Валидный HTML5 и CSS3
- Семантическая разметка для accessibility
