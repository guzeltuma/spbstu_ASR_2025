# Промпт для генерации лендинга DataInsight Pro

Создай одностраничный лендинг для сервиса DataInsight Pro - облачной платформы для анализа данных для малого бизнеса.

## Структура лендинга (обязательные блоки):

1. **Hero Section (главный блок)**
   - Привлекательный заголовок "Раскройте потенциал ваших данных"
   - Краткое описание сервиса (2-3 предложения)
   - Кнопка CTA "Начать анализ бесплатно"
   - Фон с градиентом или изображением, связанным с аналитикой

2. **Преимущества (Features Section)**
   - 4-6 ключевых преимуществ с иконками
   - Каждый блок: иконка + заголовок + краткое описание

3. **Как это работает (How It Works Section)**
   - 3-4 шага процесса использования сервиса
   - Нумерованные блоки с описанием

4. **Отзывы клиентов (Testimonials Section)**
   - 2-3 отзыва от вымышленных клиентов
   - Фото, имя, должность, цитата

5. **Цены (Pricing Section)**
   - 3 тарифных плана (Базовый, Профессиональный, Корпоративный)
   - Для каждого: название, цена, список функций, кнопка "Выбрать"

6. **Форма контакта (Contact Section)**
   - Форма с полями: имя, email, компания, сообщение
   - Кнопка отправки
   - Контактная информация (email, телефон)

7. **Footer**
   - Копирайт, ссылки на соцсети, политика конфиденциальности

## Дизайн и стиль:
- **Цветовая схема**: Основной синий (#0066CC), белый фон, акценты серого
- **Шрифты**: Современные sans-serif (Arial, Helvetica или аналогичные)
- **Стиль**: Минималистичный, профессиональный, ориентированный на бизнес
- **Акценты**: Синие кнопки CTA, hover-эффекты
- **Изображения**: Использовать плейсхолдеры или простые геометрические фигуры

## Технические требования:
- Полностью адаптивный дизайн (mobile-first)
- Использовать только HTML, CSS, JavaScript (без фреймворков)
- Валидный HTML5 и CSS3
- Семантическая разметка
- Минимум интерактивности: hover-эффекты, плавная прокрутка к якорям
- Форма должна иметь базовую валидацию

## Дополнительные элементы:
- Навигационное меню в шапке с якорными ссылками
- Анимации появления блоков при прокрутке (опционально)
- Модальное окно для демонстрации (опционально)

Создай три отдельных файла: index.html, styles.css, script.js
